# Halloween 2025 - Spider Haunted House

**Theme:** Cult of Arachne
**Date:** October 2025
**Status:** ✅ Complete

## Event Overview

A 6-chamber spider-themed haunted house experience where visitors explore an abandoned cult lair.

### Chambers

1. **Candy Room** - Inviting entry chamber with false sense of security
2. **Spider Web Tunnel** - Disorienting transition with window scare and projections
3. **Trash Room** - Desiccated victim disposal area
4. **Victim Room** - Fresh cocoons with twitching bodies
5. **Hatchery** - Spider egg chamber with scavenger hunt
6. **Spider's Lair** - Final confrontation with mama spider
